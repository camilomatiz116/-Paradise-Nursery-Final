# Paradise Nursery Shop

A premium online plant shop project built with React and Redux.

## ✨ Features
- **Landing Page**: Immersive entrance with glassmorphism and animations.
- **Interactive Catalog**: Explorer 18 diverse plants with detailed descriptions.
- **Smart Shopping Cart**: Real-time quantity updates and subtotal calculations.
- **Redux State Management**: Robust cart logic using Redux Toolkit.
- **Responsive Design**: Optimized for all screen sizes.

## 🛠️ Project Structure
- `src/App.jsx`: Main entry point and view switcher.
- `src/ProductList.jsx`: The catalog orchestrator.
- `src/PlantCard.jsx`: Reusable component for plant items.
- `src/CartItem.jsx`: Shopping cart management interface.
- `src/plantsData.js`: Centralized catalog data.
- `src/CartSlice.jsx`: Redux logic for cart operations.

## 🚀 Getting Started
1. `npm install`
2. `npm run dev`
